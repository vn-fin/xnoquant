# XNO Quant 

This repository serves as a testing ground for AI-driven quantitative analysis and engineering. It contains various projects and templates designed to evaluate different levels of engineering expertise.

## Project Structure

- **`fullstackAI/`**: A senior developer entrance test template for building an end-to-end market data system.
- **`intern/`**: Specific challenge descriptions for Intern/Junior roles (AI, Backend, Data Engineering).
- **`.codex`**: Project configuration for AI-assisted development.

## AI Engineering Testing

This repository is actively used to test and refine AI engineering capabilities. We encourage the use of AI tools (ChatGPT, Claude, GitHub Copilot, etc.) in all tasks, provided that the usage is transparently documented.

---

### Senior Entrace Test

If you are here for the senior developer entrance test, please navigate to the [fullstackAI/REQUIREMENTS.md](./fullstackAI/REQUIREMENTS.md) for detailed instructions and evaluation criteria.

The test focuses on:
- Large-scale Data Pipelines (40%)
- Professional Backend Architecture in Go (25%)
- AI Engineering & Signal Generation (20%)
- Frontend Visualization (10%)
- DevOps local deployment (5%)

---

### Community and Data

We believe in engineering freedom. Candidates and contributors are encouraged to:
- Choose their own databases and frameworks.
- Use any data sources (TradingView, SSI, Yahoo Finance, Reuters, etc.).
- Leverage AI agents to accelerate development while maintaining code quality.
